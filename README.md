# LineFollower
A QTR sensor based Line follower made for AXIS VNIT 24

# Pins declarations
  Calibration button: A6
  Start button: A7
  Left motor forward: 6
  Left motor enable: 9
  Right motor forward: 3
  Right motor enable: 5

  QTR Sensors (From right to left of array): 8, A0, A1, A2, A3, A4, A5, 7
